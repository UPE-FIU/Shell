# :shell: Shell

- Dashboard for Shellhacks 2019

## Available scripts

`yarn dev` - starts CRA dev server
`yarn start` - starts production server with pm2
`yarn stop` - stops pm2 process
`yarn restart` - restarts pm2 process
`yarn build` - bundle up react
`test` - Runs CRA tests
